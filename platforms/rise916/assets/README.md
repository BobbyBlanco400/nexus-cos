# Assets Directory

This directory contains platform-specific assets for Rise916.

## Asset Categories

- **branding/** - Logos, brand guidelines, color schemes
- **images/** - Platform images, artist photos, event images
- **videos/** - Promotional videos, artist performances
- **audio/** - Audio samples, background music
- **3d-models/** - 3D models for virtual stage

## Branding Assets Needed

- rise916_logo.svg - Platform logo
- Platform color palette:
  - Primary: #FF6F61 (Coral/Orange)
  - Secondary: #1B1B1B (Dark Gray/Black)
  - Accent: #FFD700 (Gold)
- Font: Montserrat (sans-serif)

## Asset Guidelines

All assets should:
- Be optimized for web delivery
- Follow the platform's branding guidelines
- Include proper attribution for artist content
- Be accessible and comply with web standards
