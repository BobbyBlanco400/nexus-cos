## üìã Operational Procedures

### Daily Operations
1. Check system health via control panel
2. Review analytics dashboards
3. Monitor resource usage
4. Check security logs

### Deployment Process
1. Run verification suite: `./nexus-ai/verify/run-all.sh`
2. If all checks pass, deploy: `./nexus-deploy.sh`
3. Monitor control panel for issues
4. Verify all services healthy

### Rollback Procedure
1. Identify failing tier
2. Stop affected tier: `docker-compose stop <tier-services>`
3. Restore last known good images
4. Restart tier: `docker-compose up -d <tier-services>`
5. Verify health before proceeding
